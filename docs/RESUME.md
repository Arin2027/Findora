# Resume Project Description

**Findora — AI-Powered Lost & Found Platform**

- Built a full-stack SaaS-style web app (React, Node.js, MongoDB) with semantic item matching using OpenAI embeddings and CLIP image similarity, achieving multi-signal scoring (text, image, geolocation).
- Implemented real-time notifications and chat via Socket.IO room architecture, JWT auth with refresh tokens, email verification, and admin analytics (Recharts).
- Deployed with Docker, CI/CD (GitHub Actions), Cloudinary media, optional Redis caching, and production-ready layered API architecture.
